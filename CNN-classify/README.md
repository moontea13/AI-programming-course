# 实验一 场景分类实践
## 实验任务
目标分类使用CIFAR10数据集，CIFAR10数据集包含60,000张32x32的彩色图片，10个类别，每个类
包含6,000张。其中50,000张图片作为训练集，10000张作为验证集。

## 实验流程：
1. 下载并加载数据集
2. 搭建卷积神经网络
3. 模型训练与评估

## 完整实验环节：
1. 基线搭建
2. 调参与学习策略（学习率、批量大小、优化器、epoch、数据增强等）
3. 消融实验
4. 多方法对比
5. 结果整理与可视化
6. 模型保存与代码管理
